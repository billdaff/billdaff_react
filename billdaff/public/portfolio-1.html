NJCourts.gov Website Redesign

Description

  • https://njcourts.gov

I was project leader for the team that redesigned and implemented the https://njcourts.gov website. The website serves over 27 million users yearly throughout the state of New Jersey, including general public, attorneys, and judges. The redesign required a new look, restructured data/content, and implementation of newer technologies. Read more in the Judiciary Times. The new site features/technologies include:

  • HTML5/CSS3 with Bootstrap to provide screen responsiveness for all devices.
  • Ajax/jQuery/JSON to allow for content to be centralized and dynamically called onto pages.
  • Implementation of ADA Standards to assist users with disabilities when accessing the site.
  • Utilization of data tables to assist in finding content through table filters.
